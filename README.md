# Mastermind

### __HOW TO RUN__

1. Java Compiler

        $ javac *.java -d .

2. Run the program with below cmd
   
        $ java mastermind.GuessNum

__OR__

outside dir

1. Java Compiler

        $ javac -cp . mastermind/*.java

2. Run the program with below cmd
3. 
        $ java -cp . mastermind.GuessNum
